<h-hero>Tag</h-hero>

A small inline pill for displaying structured metadata like subtypes, data types, or categories.

[:hero]

# Syntax

```markdown
[tag: *Label*]
```

The label can be any short text. Tags can be **inline elements** sting within a sentence or a list item, not as standalone blocks.

## Example

```markdown
The dataset contains [tag: Genomics] and [tag: Proteomics] data from Germany.
```

The dataset contains [tag: Genomics] and [tag: Proteomics] data from Germany.
